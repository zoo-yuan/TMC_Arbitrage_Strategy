# A股总市值与GDP比值可视化系统

## 简介

本系统展示大陆股票市场（A Share）总市值与GDP的比值曲线，支持实时数据更新。

## 功能特性

- **实时市值数据**：每30秒自动刷新，数据来源为东方财富
- **周期选择**：支持1月、6月、1年、3年、5年、10年、全历史等周期
- **GDP比例线**：6条参考线（0.3×、0.5×、0.6×、0.8×、1.0×、1.17× GDP）
- **交互提示**：鼠标悬停显示当日总市值、当年GDP、实时比例

## GDP比例参考线说明

| 系数 | 含义 |
|------|------|
| 0.3×GDP | 较低水平 |
| 0.5×GDP | 中等偏低 |
| 0.6×GDP | 中等水平 |
| 0.8×GDP | 中等偏高 |
| 1.0×GDP | 100% GDP（红色虚线）|
| 1.17×GDP | 较高（历史峰值区）|

## 启动方式

### 1. 启动代理服务（获取实时数据）

```bash
node server/proxy.cjs
```

服务运行在 `http://localhost:3000`

### 2. 启动前端服务

```bash
npm run dev
```

访问 `http://localhost:5173`

## 数据说明

### 市值数据
- **来源**：东方财富行情API
- **更新频率**：每30秒
- **范围**：沪深两市全部A股

### GDP数据
- **历史年份**：国家统计局实际公布值
- **2026年**：2025年GDP × 5%增长目标 ≈ 140万亿（预估）

### 注意事项
- 实时市值与部分软件（如通达信）可能有差异，原因包括计算口径不同
- GDP比例为预估，仅供参考

## 技术栈

- React 18 + TypeScript
- ECharts 5（图表）
- TailwindCSS（样式）
- Express（代理服务）
