# A股总市值与GDP比值可视化系统

## 简介

本系统展示大陆股票市场（A Share）总市值与GDP的比值曲线，支持实时数据更新、指数叠加、个股对比、ETF份额追踪等功能。

## 功能特性

- **实时市值数据**：每30秒自动刷新，数据来源为上交所+深交所官方API
- **历史数据**：基于真实指数K线回算，可追溯至1996年
- **周期选择**：支持1月、6月、1年、3年、5年、10年、全历史、自定义日期范围
- **GDP比例线**：6条参考线（0.3×、0.5×、0.6×、0.8×、1.0×、1.17× GDP），2000年前不绘制
- **指数叠加**：上证指数、沪深300、中证500/1000/2000/A500、中证红利，自动量纲转换
- **个股对比**：搜索并叠加任意A股/港股的价格和PE曲线
- **平均股价**：全市场平均股价（总市值÷总股本），可选叠加
- **ETF300份额**：华泰柏瑞沪深300ETF季度份额变动，阶梯线显示，用于跟踪中央汇金操作
- **交互提示**：鼠标悬停显示详细数据

## GDP比例参考线说明

| 系数 | 含义 |
|------|------|
| 0.3×GDP | 较低水平 |
| 0.5×GDP | 中等偏低 |
| 0.6×GDP | 中等水平 |
| 0.8×GDP | 中等偏高 |
| 1.0×GDP | 100% GDP（红色虚线）|
| 1.17×GDP | 较高（历史峰值区）|

## 启动方式

### 1. 启动代理服务（获取实时数据）

```bash
node server/proxy.cjs
```

服务运行在 `http://localhost:3000`，提供8个API端点。

### 2. 启动前端服务

```bash
npm run dev
```

访问 `http://localhost:5173`

## 数据说明

### 市值数据
- **来源**：上交所+深交所官方API（优先），东方财富（北证备用）
- **历史数据**：基于上证指数+深证综指真实K线回算
- **更新频率**：每30秒
- **范围**：沪深北三市全部A股

### GDP数据
- **历史年份**：国家统计局实际公布值
- **2025年**：2024年GDP × 1.05 ≈ 141.6万亿（预估）
- **2026年**：2024年GDP × 1.05² ≈ 148.6万亿（预估）

### ETF300份额数据
- **来源**：东方财富基金F10（华泰柏瑞沪深300ETF, 510300）
- **频率**：季度（基金季报）
- **范围**：2012年基金成立至今（约59个季度）
- **用途**：通过份额变动跟踪中央汇金公司A股操作节奏

### 注意事项
- 实时市值与部分软件（如通达信）可能有差异，原因包括计算口径不同
- GDP比例为预估，仅供参考
- 个股PE为动态估算（基于当前PE按股价比例推算历史PE）

## 技术栈

- React 19 + TypeScript (strict)
- ECharts 6（图表）
- Tailwind CSS 4（样式）
- Vite 8（构建）
- Express 5（代理服务）

## API端点

| 端点 | 说明 |
|------|------|
| `GET /api/total-market-cap` | 实时总市值（沪深北分项） |
| `GET /api/market-cap/history?limit=N` | 市值历史数据 |
| `GET /api/index/kline?symbol=xxx&limit=N` | 指数K线 |
| `GET /api/stock/search?keyword=xxx` | 股票搜索 |
| `GET /api/stock/kline?secid=x.xxx&limit=N` | 个股K线 |
| `GET /api/stock/realtime?secid=x.xxx` | 个股实时数据 |
| `GET /api/avg-stock-price/history?limit=N` | 平均股价历史 |
| `GET /api/etf/shares-history?code=510300` | ETF份额历史 |
