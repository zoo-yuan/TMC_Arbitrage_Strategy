# BTC 彩虹图（Bitcoin Rainbow Chart）计算逻辑

## 概述

BTC 彩虹图是一种基于**对数回归（Logarithmic Regression）**的长期价格分析工具，用于捕捉比特币随时间递减的增长率。该模型由 Bitcointalk 用户 "Trolololo" 于 2014 年首次提出，后经多次优化调整。

---

## 核心数学公式

### 基础回归方程

```
Price = 10^(a · ln(x) + b)
```

或转换为对数形式：

```
log₁₀(Price) = a · ln(x) + b
```

### 参数说明

| 参数 | 含义 |
|------|------|
| **x** | 距离 2009 年 1 月 9 日（创世区块）的周数 |
| **ln(x)** | 时间的自然对数 |
| **a, b** | 回归系数（通过历史数据拟合确定） |

---

## 具体公式变体

### 1. Trolololo 原版（2014）

```
Price = 10^(3.109 · ln(x) - 8.164)
```

- **来源**: Bitcointalk 论坛原始帖子（2014年10月22日）
- **特点**: 最早的对数回归模型，奠定了彩虹图的理论基础

### 2. Blockchaincenter 版（Über Holger）

```
Price = 10^(2.907 · ln(x) - 19.493)
```

- **来源**: blockchaincenter.net
- **用途**: 现代彩虹图上层红色带的基础

### 3. 替代回归模型

```
Price = 10^(2.652 · ln(x) - 18.163)
```

- **来源**: Coin Bureau 分析
- **应用**: 部分第三方工具的基准曲线

## 彩虹带生成逻辑

### 生成步骤

1. **计算基准曲线**：使用上述公式计算中心回归线
2. **添加偏移量**：在基准线上下通过固定倍数生成 9 条彩色带

### 带位计算公式

```javascript
// 上带（过热区）
UpperBand = BasePrice × (1.6^n)  // n = 1, 2, 3, 4

// 下带（累积区）
LowerBand = BasePrice × (0.7^n)  // n = 1, 2, 3, 4
```

### 颜色分区含义

| 颜色 | 区间 | 含义 | 操作建议 |
|------|------|------|----------|
| 🔵 深蓝 | -2σ | 极度低估 | 强烈买入 |
| 🟢 绿色 | -1σ ~ -0.5σ | 低估 | 累积 |
| 🟡 黄色 | ±0.1σ | 公允价值 | 持有 |
| 🟠 橙色 | +0.5σ ~ +1σ | 高估 | 考虑减仓 |
| 🔴 红色 | +2σ | 极度高估 | 卖出 |

## 模型的经济学假设

### 1. 幂律增长（Power-Law Growth）

比特币长期价格遵循幂律分布：

```
Price ∝ x^α
```

其中 α 为衰减系数，反映增长率随时间递减。

### 2. 对数尺度压缩

- **价格对数化**：将指数级增长转换为线性趋势
- **时间对数化**：压缩早期巨大波动，突出长期规律

### 3. 市场成熟效应

| 阶段 | 时间 | 增长倍数 | 特征 |
|------|------|----------|------|
| 早期 | 2009-2013 | 1000×+ | 极高波动 |
| 成长期 | 2014-2020 | 100× | 机构入场 |
| 成熟期 | 2021+ | 10× | 主流 adoption |

---

## 模型有效性

### 拟合度指标

- **R² ≈ 0.95**：模型解释约 95% 的长期价格方差
- **时间跨度**：2009 年至 2026 年（17年数据）

### 模型失效事件

| 时间 | 事件 | 影响 |
|------|------|------|
| 2018 年 | 跌破下带 | 模型首次失效，触发 V2 调整 |
| 2022 年 11 月 | 跌破 $16,000 | 跌破原模型下限，促使参数重新校准 |
