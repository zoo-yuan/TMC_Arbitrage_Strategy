# 大陆股票市场总市值与GDP比值可视化

## 1. 项目概述

**项目名称**: A股总市值与GDP比值可视化系统
**核心功能**: 展示任意周期内的大陆股票市场总市值曲线，并叠加6条不同GDP比例的水平参考线
**交互**: 鼠标悬停显示当日总市值、当年GDP、实时比例

---

## 2. 数据说明

### 2.1 市值数据
- **来源**: 聚宽/Tushare API（先以模拟数据演示）
- **频率**: 日频
- **品种**: A股总市值（沪深两市合计）

### 2.2 GDP数据
| 年份 | 数据来源 |
|------|----------|
| 历史年份 | 国家统计局实际公布值 |
| 2026 | 2025年GDP × 2026增长目标(5%) ≈ 140万亿 |

### 2.3 GDP比例参考线
6条水平线 = 系数 × 当年GDP

| 系数 | 含义 |
|------|------|
| 0.3 | 较低水平 |
| 0.5 | 中等偏低 |
| 0.6 | 中等水平 |
| 0.8 | 中等偏高 |
| 1.0 | 100% GDP |
| 1.17 | 较高（历史峰值区） |

---

## 3. 功能需求

### 3.1 图表展示
- **主曲线**: 总市值变动曲线（日频）
- **6条水平线**: 不同系数 × 当年GDP
- **坐标轴**: 左轴为市值（万亿元），右轴可显示GDP

### 3.2 周期选择
- 预设: 近1月、近6月、近1年、近3年、近5年、近10年、全历史
- 自定义日期范围

### 3.3 交互提示（鼠标悬停）
悬停显示:
```
日期: 2024-03-15
总市值: XX.XX 万亿元
当年GDP: XX.XX 万亿元
实时比例: XX.XX% (= 总市值/当年GDP)
```
