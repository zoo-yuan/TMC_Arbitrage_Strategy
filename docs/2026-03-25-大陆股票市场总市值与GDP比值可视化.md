# 大陆股票市场总市值与GDP比值可视化

## 1. 项目概述

**项目名称**: A股总市值与GDP比值可视化系统
**核心功能**: 展示任意周期内的大陆股票市场总市值曲线，并叠加6条不同GDP比例的水平参考线
**数据范围**: 1996年至今（约8000个交易日），基于真实指数K线回算
**交互**: 鼠标悬停显示当日总市值、当年GDP、实时比例；支持数据缩放

---

## 2. 数据说明

### 2.1 市值数据
- **来源**: 上交所+深交所官方API（实时）+ 北证（东方财富）
- **历史数据**: 基于上证指数和深证综指真实K线回算（1996年至今）
- **频率**: 日频
- **范围**: 沪深北三市全部A股

### 2.2 GDP数据
| 年份 | 数据来源 |
|------|----------|
| 2000-2024 | 国家统计局实际公布值 |
| 2025 | 2024年GDP × 1.05 ≈ 141.6万亿（预估）|
| 2026 | 2024年GDP × 1.05² ≈ 148.6万亿（预估）|

### 2.3 GDP比例参考线
6条水平线 = 系数 × 当年GDP（2000年前不绘制）

| 系数 | 含义 |
|------|------|
| 0.3 | 较低水平 |
| 0.5 | 中等偏低 |
| 0.6 | 中等水平 |
| 0.8 | 中等偏高 |
| 1.0 | 100% GDP（红色虚线）|
| 1.17 | 较高（历史峰值区） |

## 3. 功能特性

### 3.1 图表展示
- **主曲线**: 总市值变动曲线（日频，含面积填充）
- **6条GDP参考线**: 不同系数 × 当年GDP
- **指数叠加**: 7个可选指数（上证、沪深300、中证500/1000/2000/A500、中证红利）
- **个股叠加**: 搜索并叠加任意A股/港股的价格和PE曲线
- **平均股价**: 全市场平均股价 = 总市值 ÷ 总股本
- **ETF300份额**: 华泰柏瑞沪深300ETF(510300)季度份额变动，跟踪中央汇金操作

### 3.2 周期选择
- 预设: 近1月、近6月、近1年、近3年、近5年、近10年、全历史
- 自定义日期范围

### 3.3 交互提示（鼠标悬停）
悬停显示:
```
日期: 2024-09-30
总市值: 84.50 万亿
当年GDP: 134.90 万亿
实时比例: 62.64%
ETF300份额: 969.2亿份 | 净资产:3975亿元
个股: 贵州茅台 ¥1680.00 PE:25.3
```

## 4. 技术方案

- **React 19** + TypeScript (strict)
- **ECharts 6** (图表)
- **Tailwind CSS 4** (样式)
- **Vite 8** (构建)
- **Express 5** (代理服务)

---

## 5. 项目结构

```
src/
├── components/
│   └── MarketValueChart.tsx   # ECharts多轴图表
├── hooks/
│   ├── useMarketData.ts       # 市值数据编排
│   └── useStockData.ts        # 个股搜索/K线/实时
├── data/
│   ├── gdpData.ts             # GDP数据 2000-2026
│   └── indexData.ts           # 指数配置与量纲转换
├── App.tsx                    # 主应用
└── main.tsx                   # 入口
server/
└── proxy.cjs                  # Express代理(8个API端点)
```
